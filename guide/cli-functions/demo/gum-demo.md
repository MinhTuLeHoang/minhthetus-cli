# Gum Interaction Demo

Demonstrates the enhanced UI capabilities provided by the `gum` tool.

## Usage
```bash
minhthetus-cli demo gum-demo
```

## Features Demonstrated
*   **Stylized Headers**: Using `gum style` with borders, colors, and padding.
*   **Interactive Choices**: Using `gum choose` to navigate a menu.
*   **User Input**: Using `gum input` to capture text from the user.
*   **Spinners**: Using `gum spin` to show background task progress.
*   **Styled Output**: Using colors and symbols for status messages.
