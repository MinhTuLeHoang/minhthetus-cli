# Prompt: Update CLI Documentation

Use the following prompt when you want the AI to update the documentation based on the latest code changes. Note that this functionality is also available as an Antigravity **skill** (`update-docs`) and can be triggered via workflow.

---

## The Prompt

"Please use your `update-docs` skill to review the current implementation of the CLI scripts in the `src/scripts` directory and update the corresponding documentation files in the `guide/cli-functions` folder."

---

## How to use
1.  **Direct Instruction**: Ask me to "update documentation using the `update-docs` skill".
2.  **Prompting**: Copy the text above and paste it into our conversation.
3.  **Workflow**: Use the automation folder if a workflow is created for this task.
